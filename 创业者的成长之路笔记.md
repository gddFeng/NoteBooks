# 第二单元

## 2.1 创业意识

### 2.2.1 创业意识的内涵

意识是人的头脑对于客观物质世界的反应，是感觉、思维等各种**心里过程的总和**，是社会的人堆客观存在的主观反映。

### 2.2.2 创业意识的内涵

意识的内涵：

* 穿也活动或过程室友创业意识和创业行为这两大方面构成的。
* 创业意识是指在创业实践活动中对创业者起**动力**作用的个性倾向。
* 创业意识集中表现了创业素质中的社会性质，支配着创业者对创业活动的**态度和行为**，并规定其态度和行为的方向、力度、层次和水平，具有较强的主观选择性和能动性，是创业素质的重要组成部分。
* 创业意识是创业者素质系统中的第一个子系统即**驱动系统**，是创业行为产生的根本前提和先导，**没有创业意识也就没有创业行为**。

创业意识的产生：

创业意识是在**创业知识、创业技能与创业经验**的基础上产生的。有没有创业知识、创业技能与创业经验做为基础，很难产生创业意识，最多只能得出一点非常浮现的创业想法或意向。因此，创业意识是**创业知识、创业技能和创业经验的进一步升华**。

### 2.2.3 创业意识的要素

创业意识是由创业需要、创业动机、创业星宇、创业理想、创业信念和创业世界观等要素组成的。

* 创业需要
  * 创业需要上升为创业动机。具有仅有创业需要不一定有创业行为，只有创业需要上升为创业动机时，创业行为才有可能发生。
* 创业动机
  * 激发创业动机的内外条件。引起创业动机的**内在条件**是创业需要；引起创业动机的**外在条件**是创业目标或对象。
  * 创业动机的作用。创业动机是一种成就动机，是竭力追求获得最佳价值和优异效益的动因。创业动机对创业行为产生促进、推动左右。创业者创业动机的形成标志着创业实践活动的即将开始，**有了创业动机，才会有创业行为**。
* 创业兴趣
  * 要把创业兴趣升华到创业爱好
* 创业理想
  * 当创业动机和创业兴趣积累到一定程度时就形成了创业理想。主要是一种**职业理想和事业理想**，而非政治理想和道德理想。
* 创业信念
  * 创业没有失败者，只有放弃者
  * 创业信念是认识、情感和意志的有机统一，是一种综合性、稳定性和持久性很强的心理品质。因此创业信念是创业者的精神支柱，是创业意识的核心部分。

## 2.2 创业精神

创业精神是指在创业者的主观世界中，那些具有**开创性**的思想、观念、个性、意志、作风和品质等

创业者的个体创业精神是指以个人力量，在个人意志的引导下，开创新事业的思想、理念和品质。

### 创业精神的层次

1. 哲学层次创业精神
2. 心理学层次创业精神
3. 行为学层次创业精神

### 创业精神的主题

创业精神是一种追求机会的行为境界，不断追求和把握机会和机遇、勇敢地面对各种各样的挑战。

创业精神是持续变革、革新、转换和引入新理念、新方法、新技术等的集中体现。创新是创业精神的核心内涵。

一是对规模的追求；二是对质量个价值的追求；三是追求持续的创新和发展

这三个主题随着创业行为深化而不断丰富新的内涵，进而不断提升创业的质量与价值并贯穿创业过程的始终。

### 创业精神的特征

* 高度综合性
* 三维整体性
* 整体先进性
* 鲜明时代性

# 第三单元

## 3.1 磨炼创业心理品质与思维

### 创业心理品质的定义

创业心理品质是指创业者在创业实践活动中表现出来的**心里过程和个性心理特征**。主要体现在创业者的独立性、敢为性’坚韧性、可执行、适应性、合作性等方面。创业心理品质是创业者在创业实践活动中克服重重困难、达成创业目标的前提和关键。

### 创业心理品质的内涵

1. 强烈的创业需要动机
2. 高度的自信心
3. 较强的自我控制力
4. 较强的冒险精神与拼搏意识
5. 较强的创新意识与创新能力
6. 高度诚信与较强协作能力

### 创业心理品质的塑造

良好的创业邻里品质是创业者创业成功的基本前提和条件。一个创业者的创业心理品质不是先天就有的，而是通过后天的实践活动、教育培养逐步形成的。

从意义上来说，每个人都有特定的创业心理品质结构，每个人都具有显性的、隐性的创业心理品质，都有相对优势、劣势的创业心理品质。

### 创业心理品质的塑造

1. 强化自我修炼
2. 学习和借鉴创业成功者的经验
3. 在创业实践中不断总结经验和教训
4. 积极参加创业心理咨询和训练

# 第四单元

## 洞察力

洞察力，是指通过表面现象精确判断出现象背后的本质的能力

敏锐的观察力是创业者必备的素质之一。有敏锐洞察力的创业者往往善于把握创业机会。

### 洞察力的本质

洞察力的本质是一种深入观察、理解和把我事物或问题的能力

洞察力是人们对个人认识、情感、行为的动机与相互关系的透彻分析

### 洞察力的意义

缺乏洞察力：往往会把组织引向停滞和萧条，瞬间的粗心大意就有可能导致失败。

具备洞察力：往往能够发现别人容易忽略甚至从未注意到的机会，并会鞭策组织保持它的优势地位。

### 创业机会

创业机会是指具有较强吸引力的、较为持久的合适的一种有利于其创业的奇迹，并最终表现在能够为客户提供有价值或增加价值的产品或服务之中。

创业机会本身是无形的，一般附着在不断发展变化的事物当中。

普遍性、偶然性、消逝性

### 创业机会识别

1. 先前经验
2. 专业知识
3. 认知因素
4. 社会关系网络
5. 创造性

### 创业者对创业机会的洞察

1. 现有市场机会与潜在市场机会
2. 行业市场机会与边缘市场机会
3. 目前市场机会与外来市场机会
4. 全面市场机会与局部市场机会

## 沟通能力

### 沟通的含义

沟通是信息凭借一定符号载体，从发送者到接受者进行传递，并获取理解的过程。
